Fokus på denna labb var Model-View-ViewModel (MVVM)-arkitektur i C#.

Byggt en quiz applikation med hjälp av XAML. Spelaren kan växla mellan vyer som syns nedanför, navigera menyer med tangentknappar och lägga till och ta bort frågor och question packs och spela quizspelet.
